# key-mouse-controller<br>
key-mouse-controller(c语言)<br>


linux下边实现键盘和鼠标控制功能。<br><br>

编译方法：<br>
    gcc -o key-mouse-controller key-mouse-controller.c `pkg-config --cflags --libs gtk+-2.0`<br>
执行方法：<br>
    sudo ./key-mouse-controller<br><br>
